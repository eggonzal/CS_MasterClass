# C# Master Class Projects

A collection of C# learning projects.

## Projects

- **FirstWinFormsApp** - Windows Forms app for suggesting optimal numeric types based on value ranges
- **GameDataParser** - Game data parsing utility
- **StarWarsPlanetsStats** - Star Wars planets statistics
- **CookieCookbook** - Cookie cookbook application
- **CustomLinkedList** - Custom linked list implementation
- **TicketsDataAggregator** - Tickets data aggregation
- **Caching** - Caching examples
- **SadBox** - Sandbox for experiments

## Requirements

- .NET 9 SDK
- Visual Studio 2022 or VS Code with C# extension

## Getting Started

```bash
# Clone the repository
git clone https://github.com/YOUR_USERNAME/CS_MasterClass.git

# Navigate to the solution
cd CS_MasterClass

# Build all projects
dotnet build

# Run a specific project
dotnet run --project FirstWinFormsApp
```
